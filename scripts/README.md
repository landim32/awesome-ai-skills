# Scripts

Build, validation, and utility scripts for the repository
(per constitution Principle II). Reusable CI/CD pipeline templates
belong under `workflows/`, not here.

- **Language**: English only.
- **Authoring**: kebab-case filenames; use the appropriate extension
  (`.ps1`, `.sh`, `.py`, `.js`) per Principle V.
