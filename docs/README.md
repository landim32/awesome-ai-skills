# Docs

Human-facing documentation: onboarding, architecture notes, decision
records, and how-to guides (per constitution Principle II).

- **Language**: bilingual — English or Portuguese (PT-BR). When a
  document exists in both languages, the English version uses `.md` and
  the Portuguese version uses `.pt-BR.md`. Code identifiers and
  technical keywords remain in English inside PT-BR documents.
- **Authoring**: UPPER_SNAKE_CASE filenames per the `doc-manager` skill
  (e.g., `DEPLOYMENT_GUIDE.md`, `GUIA_DE_DEPLOY.pt-BR.md`). The `analyst`
  agent is the canonical author for files in this folder.

Authoritative source: [.specify/memory/constitution.md](../.specify/memory/constitution.md).
