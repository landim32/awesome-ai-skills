# Commands

Named, invokable commands (e.g., slash commands) with defined inputs,
outputs, and side effects (per constitution Principle II).

- **Language**: English only.
- **Authoring**: kebab-case filenames; each command is self-contained;
  inputs, outputs, and side effects are documented explicitly
  (Principle V).
